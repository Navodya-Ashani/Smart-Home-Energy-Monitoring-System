# Smart Home Energy Monitoring and Control System

## Overview
This project uses ESP32 and Blynk IoT to monitor and control home energy consumption.

## Features
- Voltage Monitoring (230V)
- Current Monitoring
- Power Calculation
- Energy (kWh) Calculation
- Remote Appliance Control
- High Power Alert Notification
- Automatic Load Shedding
- Blynk Mobile Dashboard

## Components
- ESP32
- Potentiometer
- LEDs
- Push Buttons
- Blynk IoT Platform

## Software
- Arduino IDE
- Wokwi Simulator
- Blynk IoT
